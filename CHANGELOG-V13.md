# CHANGELOG V13

## H2 접두어 스타일 수정

- 기존 `post-h2-prefix`의 알약형 배지 디자인을 제거했습니다.
- 접두어의 배경, 보더, 보더 라운드, 그림자를 제거했습니다.
- 접두어는 사이트 포인트 컬러(`#2f715f`)와 밑줄 형태로 표시되도록 변경했습니다.
- 접두어 폰트 크기를 고정 px가 아닌 `0.95em`으로 변경해 `post-h2-text`와 자연스럽게 비슷한 비율로 보이도록 조정했습니다.
- H2 내부 구조를 flex 배치가 아닌 일반 텍스트 흐름에 가깝게 변경해 `위치 접근성: 제목` 형태가 한 문장처럼 이어지도록 했습니다.
- 모바일에서도 접두어가 과하게 분리되지 않고 H2 제목 안에서 자연스럽게 이어지도록 조정했습니다.
