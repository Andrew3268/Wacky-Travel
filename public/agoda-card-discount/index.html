사용 방법

URL CHECK 아고다 홈페이지의 호텔 URL 또는 앱에서 공유한 /sp/ 링크를 입력할 수 있습니다.
  1. 01
    아고다 홈페이지에 접속합니다. 아래 버튼을 눌러 아고다 홈페이지를 열어주세요. 아고다에서 호텔 찾기
  2. 02
    예약할 호텔 페이지로 이동합니다.검색 결과에서 예약할 호텔을 선택해 해당 호텔 페이지를 열어주세요.
  3. 03
    주소창의 호텔 페이지 URL을 복사합니다.홈페이지 주소 또는 앱에서 공유한 https://www.agoda.com/sp/... 링크를 복사하세요.
  4. 04
    복사한 주소를 붙여넣고 할인 혜택을 확인합니다.첫 화면 입력창에 주소를 붙여넣고 사용할 결제수단을 선택하면 전용 할인 링크가 만들어집니다.

앱에서 공유한 /sp/ 링크는 자동으로 호텔 홈페이지 주소로 변환됩니다. 검색 결과 페이지가 아닌 개별 호텔 링크를 입력해주세요.
